# Batch versus Online Centering

Batch Centering: 
  In batch processing, the entire dataset is loaded into memory, and the mean is computed from the complete data. This technique is straightforward, but it may not be ideal for very large datasets.
  
Online Centering: 
  Online or incremental centering methods update the mean as new data arrives. This is particularly useful in streaming data applications or when processing large datasets that cannot be loaded entirely into memory. Algorithms such as the exponential moving average can be used for online mean computation [2](https://en.wikipedia.org/wiki/Moving_average).

| Centering Method | Pros                                | Cons                                          |

|------------------|-------------------------------------|-----------------------------------------------|
| Batch Centering  | Simplicity; works for all dataset sizes if in memory  | Not suitable for extremely large datasets     |
| Online Centering | Memory efficient; real-time updates | More complex; requires incremental algorithms |

Each approach has its merits and should be chosen based on the specific analysis context, especially in scenarios with resource constraints or real-time processing demands.
