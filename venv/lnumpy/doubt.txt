in Numpy(np), axis tells which direction you collapse(reduce)
when doing an operation like mean(), std(), or more...

example
data = np.array([[7,4,4],[8,2,2]])
shape = (2, 3)

axis = 0 mean? row 1D
axis = operate down the rows(vertically)
combine the value row-wise
result column-wise
axis = 0 = collapse rows = result per column
